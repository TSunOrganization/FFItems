# 📖 About  


``` FFItems is a web application that allows users to easily search and browse Garena Free Fire in-game items.  ```

## You can find

- ✅ Item IDs  
- ✅ Item descriptions  
- ✅ Easy browsing experience  

This tool is especially useful for developers, gamers.
